# 네온 브레이커 (Neon Breaker)

## 📁 폴더 구조
```
neon-breaker/
├── index.html          # 메인 게임
├── neon_admin.html     # 관리자 페이지
└── media/
    ├── thumbs/         # 캐릭터 썸네일
    │   ├── char_1.jpg
    │   ├── char_2.jpg
    │   └── ...
    └── stages/         # 스테이지 클리어 이미지/영상
        ├── r1_s1.jpg   # 라운드1 스테이지1
        ├── r1_s2.mp4   # 라운드1 스테이지2
        └── ...
```

## 🚀 배포 방법
1. GitHub에 업로드
2. Vercel 연결 → 자동 배포

## 📷 미디어 파일명 규칙
- 캐릭터 썸네일: `char_1.jpg`, `char_2.png` ...
- 스테이지: `r1_s1.jpg` (라운드1 스테이지1)
